# terraform-aws-multi-instance-infra
Automated AWS infrastructure with Terraform: custom VPC, public/private subnets, EC2 (Linux &amp; Windows), security groups, CloudWatch alarms + SNS notifications. Designed with banking‑grade reliability and compliance mindset

## 🏦 Banking Context
Working in the banking domain, I designed this project with reliability, compliance, and proactive monitoring in mind — reflecting how financial institutions can leverage IaC for secure cloud operations.
